# Full Stack Web Development

A personal learning repository documenting my journey through full-stack web development. Contains hands-on exercises, mini-projects, and code written while working through Angela Yu's Complete Web Development Bootcamp on Udemy.

## Topics Covered

- **HTML** — document structure, semantic elements, forms, self-closing tags
- **CSS** — selectors, layouts, Flexbox, Grid, responsive design
- **JavaScript** — DOM manipulation, events, ES6+
- **Node.js & Express** — server-side development, REST APIs
- **React** — components, hooks, state management
- **Databases** — SQL and NoSQL fundamentals
